### Setup Instructions

1. Clone the repository:
    ```bash
    git clone https://github.com/username/project-name.git
    cd splinter-backend
    ```

2. Create a virtual environment:
    ```bash
    python -m venv venv
    ```

3. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

4. Run the app:
    ```bash
    python main.py
    ```
