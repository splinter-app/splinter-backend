### Setup Instructions

1. Clone the repository:
    ```bash
    git clone https://github.com/splinter-app/splinter_backend.git
    cd splinter_backend
    ```

2. Create a virtual environment:
    ```bash
    python -m venv venv
    ```

3. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

4. Run the app:
    ```bash
    uvicorn main:app --reload
    ```
